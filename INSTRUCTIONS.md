# Настройка Cloudflare Worker для анкеты гостя

Эта инструкция за 5–7 минут подключит вашу анкету на сайте к Telegram-боту.
Никаких платежей и привязки карты не требуется — используется бесплатный план Cloudflare.

---

## Шаг 1. Регистрация на Cloudflare

1. Откройте [https://dash.cloudflare.com/sign-up](https://dash.cloudflare.com/sign-up)
2. Зарегистрируйтесь по email (подтвердите почту по ссылке из письма)
3. Войдите в личный кабинет — [https://dash.cloudflare.com](https://dash.cloudflare.com)

---

## Шаг 2. Создание Worker'а

1. В левом меню найдите **Workers & Pages**
2. Нажмите **Create** (или **Create application** → вкладка **Workers**)
3. Нажмите **Create Worker**
4. Дайте имя, например: `wedding-rsvp` (можно любое — оно станет частью адреса)
5. Нажмите **Deploy** — Cloudflare создаст шаблонный Worker

---

## Шаг 3. Вставка кода

1. После деплоя нажмите **Edit code** (или **Configure Worker** → **Edit code**)
2. Откроется онлайн-редактор с шаблонным кодом — выделите всё (Ctrl+A) и удалите
3. Откройте файл **`worker.js`** (приложен рядом с этой инструкцией), скопируйте весь его код
4. Вставьте в редактор Cloudflare
5. Нажмите **Save and Deploy** (или **Deploy**) в правом верхнем углу

Токен бота и chat_id уже вписаны в код — менять ничего не нужно, если вы не создавали другого бота.

---

## Шаг 4. Получение адреса Worker'а

После деплоя Cloudflare покажет адрес вида:

```
https://wedding-rsvp.ВАШ-НИКНЕЙМ.workers.dev
```

Скопируйте этот адрес целиком — он понадобится на следующем шаге.

---

## Шаг 5. Подключение адреса к сайту

1. Откройте файл `index.html` вашего сайта в любом текстовом редакторе
2. Найдите строку (она находится в самом начале блока `<script>` в самом низу файла):

```js
const WORKER_URL = "https://YOUR-WORKER-NAME.YOUR-SUBDOMAIN.workers.dev";
```

3. Замените её на ваш реальный адрес из Шага 4, например:

```js
const WORKER_URL = "https://wedding-rsvp.ivanov123.workers.dev";
```

4. Сохраните файл

---

## Шаг 6. Проверка работы

1. Откройте сайт (`index.html`) в браузере
2. Прокрутите до блока «Анкета гостя»
3. Заполните тестовую заявку и нажмите **Отправить**
4. Сообщение должно появиться у вас в Telegram от бота в течение нескольких секунд

Если сообщение не пришло — см. раздел «Возможные проблемы» ниже.

---

## Важно: активация бота

Если вы ещё не написали вашему боту хотя бы одно сообщение в Telegram — сделайте это сейчас:
1. Найдите вашего бота в Telegram по имени, которое вы дали при создании через @BotFather
2. Нажмите **Start** или напишите любое сообщение

Без этого шага бот не может присылать вам уведомления, даже если всё остальное настроено правильно.

---

## Возможные проблемы

**Анкета пишет «Не удалось отправить»**
- Проверьте, что `WORKER_URL` в `index.html` указан без ошибок и без лишних пробелов
- Проверьте в Cloudflare Dashboard → ваш Worker → вкладка **Logs**, не было ли ошибки при запросе

**В Telegram ничего не приходит, хотя сайт показал «Спасибо, ответ получен»**
- Убедитесь, что вы писали боту хотя бы раз (см. выше)
- Проверьте, что `CHAT_ID` в коде Worker'а указан правильно (`1801013206`)

**Хочу проверить Worker напрямую, без сайта**
Можно отправить тестовый запрос через сайт [https://reqbin.com](https://reqbin.com) или из терминала:
```bash
curl -X POST https://ВАШ-WORKER-АДРЕС \
  -H "Content-Type: application/json" \
  -d '{"name":"Тест Тестов","attendance":"Да, с удовольствием!","phone":"+7 900 000-00-00"}'
```
Если всё настроено верно, придёт сообщение в Telegram и ответ `{"ok":true}`.

## Если захотите создать другого бота или изменить chat_id в будущем

Откройте код Worker'а в Cloudflare (Workers & Pages → ваш Worker → Edit code) и поменяйте значения в самом начале файла:

```js
const BOT_TOKEN = "новый_токен_от_BotFather";
const CHAT_ID   = "новый_chat_id";
```

Сохраните и нажмите **Deploy** — изменения применятся мгновенно, без необходимости менять что-либо на сайте.
